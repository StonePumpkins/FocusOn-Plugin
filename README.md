# 专注屏蔽器 Pro (FocusOn Plugin)

一款 Chrome 扩展程序，支持两种屏蔽模式和设置锁定功能，帮助你保持专注。

## 功能特性

- **完全封禁模式**：在设定时段内，被屏蔽的网站完全无法访问
- **延迟进入模式**：打开被屏蔽网站后需等待 30 秒，三思而后行
- **设置锁定**：保存设置后锁定到结束时间，无法中途修改
- **防绕过设计**：刷新页面不会重置 30 秒倒计时
- **跨天时段支持**：可设置跨天的屏蔽时段（如 22:00 - 02:00）

## 文件结构

## 安装方法

1. 打开 Chrome，地址栏输入 `chrome://extensions`
2. 开启右上角 **开发者模式**
3. 点击 **"加载已解压的扩展"**
4. 选择本项目文件夹（`focus-blocker`）

## 使用说明

1. 点击浏览器右上角的扩展图标
2. 输入要屏蔽的网站（每行一个，无需 http）
3. 设置屏蔽时段（开始时间 - 结束时间）
4. 选择屏蔽模式：
   - 完全封禁：时段内网站直接打不开
   - 部分封禁：需等待 30 秒后才能进入
5. 点击 **保存** 或 **保存并锁定**

### 关于锁定功能

- 点击"保存并锁定"后，设置将被锁定到结束时间
- 锁定期间无法修改任何设置
- 弹窗会显示解锁倒计时

## 注意事项

- 首次安装后需刷新已有标签页才能生效
- 延迟模式下刷新页面不会重置 30 秒倒计时
- 锁定功能需要自律配合，无法绕过
